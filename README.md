# MindAR WebAR Project Template

Template ini disediakan untuk projek WebAR menggunakan MindAR & A-Frame.

### Cara Guna:
1. Letakkan fail marker `.mind` dalam folder `assets/`.
2. Letakkan model 3D `.glb` dalam folder `assets/`.
3. Pastikan JS (`aframe.min.js` dan `mindar-image-aframe.prod.js`) ada di folder `js/`.
4. Upload semua folder & fail ke GitHub repository.
5. Aktifkan GitHub Pages untuk branch `main`.
6. Buka URL HTTPS yang diberikan untuk scan marker dan lihat AR.
